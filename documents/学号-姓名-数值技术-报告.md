## 一、选题背景与意义

本文选择“数字经济背景下全国博物馆数量增长与文旅产业复苏”作为研究主题。该主题属于数字经济与科技创新、公共文化服务和文旅融合交叉方向。博物馆不仅是收藏、展示和教育机构，也是城市公共文化服务体系的重要组成部分。近年来，随着文化强国建设、文化数字化战略和智慧文旅建设不断推进，博物馆的功能逐渐从传统展陈扩展到线上预约、智慧导览、数字展厅、沉浸式体验、文创消费和城市旅游传播等场景。

从国家发展角度看，博物馆数量增长反映了公共文化设施供给扩张，也体现了国家对文化资源保护、公共文化服务均等化和文旅融合发展的重视。从民生角度看，博物馆数量增加意味着居民能够获得更多文化空间和教育资源。从数字经济角度看，博物馆并不是孤立的线下场馆，而是文化资源进入数字平台、连接游客需求并转化为文旅消费的重要入口。

因此，本文以全国博物馆数量、国内游客和国内旅游总花费为核心指标，研究文化供给扩张与文旅需求恢复之间的关系。本文关注的问题是：在数字经济背景下，博物馆供给持续增长是否能够重新转化为游客恢复和消费增长？2020 年后的文旅需求冲击能否通过数值方法被识别？2023 年的恢复是否已经回到正常增长趋势？这些问题既有现实意义，也适合使用数值微分、最小二乘拟合和误差分析等课程方法进行研究。

## 二、数据获取与预处理

本文数据主要来自中华人民共和国国家统计局国家数据平台及《中国统计年鉴》公开指标。选取 2014-2023 年三个年度指标：全国博物馆数量、国内游客、国内旅游总花费。其中，博物馆数量单位为“个”，国内游客单位统一为“百万人次”，国内旅游总花费单位统一为“亿元”。

原始数据如下表所示：

| 年份 | 博物馆数量（个） | 国内游客（百万人次） | 国内旅游总花费（亿元） | 阶段 |
|---|---:|---:|---:|---|
| 2014 | 3658 | 3611 | 30311.9 | 增长期 |
| 2015 | 3852 | 4000 | 34195.1 | 增长期 |
| 2016 | 4109 | 4440 | 39390.0 | 增长期 |
| 2017 | 4721 | 5001 | 45660.9 | 增长期 |
| 2018 | 4918 | 5539 | 51278.3 | 增长期 |
| 2019 | 5132 | 6006 | 57251.0 | 高点 |
| 2020 | 5446 | 2879 | 22286.3 | 冲击 |
| 2021 | 5772 | 3246 | 29191.0 | 修复 |
| 2022 | 6565 | 2530 | 20444.0 | 波动 |
| 2023 | 6833 | 4891 | 49133.1 | 修复 |

数据预处理包括四个步骤。

第一，统一口径。将三个指标对齐到 2014-2023 年共同年度区间，形成等距时间序列。由于数据为年度数据，后续数值微分中的步长取 `h = 1`。

第二，保留并标注 2020 年冲击。2020 年游客和旅游消费的下降不是普通噪声，而是公共卫生事件导致的真实社会冲击。如果简单删除该点，会弱化文旅复苏这一研究主题。因此本文保留 2020 年数据，并将其作为结构性冲击点进行解释。

第三，构造指数化指标。以 2014 年为基期，构造博物馆指数、游客指数和旅游消费指数：

```text
指数 = 当年指标值 / 2014 年指标值 × 100
```

指数化处理可以消除不同单位和数量级的影响，使不同指标能够在同一尺度下比较。

第四，构造转化效率指标。为了体现数字经济背景下“文化供给向游客和消费转化”的过程，本文构造以下派生指标：

```text
单馆游客量 = 国内游客 / 博物馆数量
人均旅游消费 = 国内旅游总花费 × 100 / 国内游客
单馆旅游消费 = 国内旅游总花费 / 博物馆数量
```

处理后发现，2019 年单馆游客量约为 1.170 百万人次/馆，2023 年下降为 0.716 百万人次/馆，说明博物馆数量持续扩张，但单馆承接游客量尚未恢复到 2019 年水平。与此同时，人均旅游消费从 2019 年的约 953.2 元/人提高到 2023 年的约 1004.6 元/人，说明游客数量恢复不足的同时，消费强度恢复较快。这为后文解释数字经济提升消费转化效率提供了数据基础。

## 三、数值建模与算法实现

### 1. 三点差分计算年度变化率

本文首先使用三点差分公式计算指标的年度边际变化。对于内部年份，采用中心差分：

```text
f'(x_i) ≈ [f(x_{i+1}) - f(x_{i-1})] / 2h
```

由于年度数据等距采样，`h = 1`。首尾年份没有两侧数据，因此分别采用前向差分和后向差分。三点差分的社会意义是把“规模变化”转化为“年度变化率”，用于判断某一指标在某一年附近是加速、减速还是受到冲击。

核心代码如下：

```python
def three_point_diff(values, h=1.0):
    diff = []
    for i, value in enumerate(values):
        if i == 0:
            diff.append((values[1] - value) / h)
        elif i == len(values) - 1:
            diff.append((value - values[-2]) / h)
        else:
            diff.append((values[i + 1] - values[i - 1]) / (2 * h))
    return diff
```

计算结果显示，2020 年附近国内游客变化率约为 `-1380.0` 百万人次/年，2023 年游客变化率约为 `+2361.0` 百万人次/年。这说明文旅需求在 2020 年受到明显冲击，而 2023 年恢复速度显著提高。

### 2. 分段最小二乘拟合

对于拟合部分，本文没有直接使用 2014-2023 年全周期线性模型，而是采用分段最小二乘。原因是 2020 年存在明显外生冲击，如果使用一条直线拟合全部年份，会把长期增长和短期冲击混在同一个斜率中，导致解释失真。

设线性拟合模型为：

```text
y = ax + b
```

最小二乘法通过最小化残差平方和确定参数：

```text
S = Σ[y_i - (a x_i + b)]²
```

本文将数据分为两个阶段：

```text
2014-2019 年：平稳增长段
2020-2023 年：冲击后恢复段
```

模型比较结果如下：

| 指标 | 全周期线性 RMSE | 全周期二次 RMSE | 分段线性 RMSE |
|---|---:|---:|---:|
| 国内游客 | 1082.5 | 1017.3 | 432.8 |
| 国内旅游总花费 | 11968.2 | 11498.7 | 5118.5 |

可以看出，分段线性模型误差明显低于全周期模型。虽然二次模型在样本内 RMSE 略低于全周期线性模型，但留一交叉验证误差更高。例如国内游客二次模型 LOOCV RMSE 为 1747.7，高于全周期线性模型的 1340.5，说明小样本下高阶模型外推不稳定。因此，本文采用分段低次最小二乘作为主要拟合方式。

核心代码如下：

```python
def least_squares_line(x_values, y_values):
    x_mean = sum(x_values) / len(x_values)
    y_mean = sum(y_values) / len(y_values)
    numerator = sum((x - x_mean) * (y - y_mean)
                    for x, y in zip(x_values, y_values))
    denominator = sum((x - x_mean) ** 2 for x in x_values)
    slope = numerator / denominator
    intercept = y_mean - slope * x_mean
    return slope, intercept
```

### 3. 反事实缺口分析

为了衡量文旅需求恢复到什么程度，本文用 2014-2019 年平稳增长趋势构造“无冲击情景”的反事实基准，并计算 2020-2023 年实际值与该基准的差距。

国内游客方面：

| 年份 | 实际值/平稳趋势 | 缺口（百万人次） |
|---|---:|---:|
| 2020 | 44.42% | -3602.5 |
| 2021 | 46.56% | -3725.6 |
| 2022 | 33.91% | -4931.6 |
| 2023 | 61.51% | -3060.7 |

国内旅游总花费方面：

| 年份 | 实际值/平稳趋势 | 缺口（亿元） |
|---|---:|---:|
| 2020 | 35.81% | -39949.8 |
| 2021 | 43.10% | -38537.0 |
| 2022 | 27.92% | -52775.9 |
| 2023 | 62.42% | -29578.7 |

该结果表明，2023 年文旅需求虽然明显恢复，但与 2014-2019 年形成的平稳增长趋势相比，仍存在较大缺口。

## 四、结果分析与价值阐释

从趋势上看，2014-2023 年全国博物馆数量从 3658 个增加到 6833 个，增长约 86.8%。这说明公共文化设施供给持续扩张，文化资源网络更加密集。但同期国内游客和旅游消费并不是单调增长，尤其在 2020 年出现明显下滑。这说明文旅产业复苏不是单纯由供给数量决定，还受到居民出行意愿、收入预期、公共卫生事件、交通条件和政策环境等因素影响。

三点差分结果进一步揭示了这种结构变化。2020 年附近游客变化率显著转负，说明需求侧受到外生冲击；2023 年游客变化率明显转正，说明文旅需求进入快速恢复阶段。相比之下，博物馆数量的年度增量始终为正，说明文化供给侧并未停止扩张。这体现了“供给持续增长、需求阶段性受阻”的特征。

分段最小二乘的结果说明，对于存在冲击的社会经济数据，不能简单使用全周期拟合。全周期模型会把平稳增长期和冲击期混在一起，导致模型解释力下降。分段低次模型既符合课程中函数逼近和最小二乘方法的思想，也能更好反映现实机制：2014-2019 年是文旅需求与文化供给共同增长阶段，2020 年后则是冲击与修复交织阶段。

反事实缺口分析进一步表明，2023 年文旅需求并没有完全回到正常增长轨道。国内游客恢复到平稳趋势的 61.51%，旅游消费恢复到平稳趋势的 62.42%。这说明如果只看同比增长，可能会高估恢复程度；如果与平稳增长趋势比较，就能看到恢复仍存在缺口。

从数字经济角度看，博物馆数量增长提供了文化供给基础，但数字经济的关键作用在于连接和转化。线上预约、智慧导览、数字展陈、短视频传播和文创电商等数字化手段，可以降低游客获取文化资源的成本，提高文化体验质量，并推动文化资源向消费场景转化。本文中人均旅游消费在 2023 年超过 2019 年，说明游客数量尚未完全恢复时，消费强度已经提升，这与文旅消费质量提升、数字化服务改善和文创消费扩展具有一定关联。

本文也存在不足。第一，年度采样间隔较大，无法反映节假日、暑期和政策变化带来的月度波动。第二，样本点只有 10 个，高阶拟合容易不稳定，因此本文只采用低次分段模型。第三，博物馆数量与游客和消费之间可能存在滞后效应，新增供给不一定在当年立即转化为游客增长。第四，本文没有直接引入线上预约人数、数字展陈访问量等数字化指标，因此数字经济部分主要作为机制解释。

综上，本文通过国家统计局相关年度数据构建时间序列，使用三点差分、分段最小二乘和反事实缺口分析，研究数字经济背景下全国博物馆数量增长与文旅产业复苏之间的关系。主要结论是：博物馆数量长期增长，说明文化供给持续扩张；2020 年文旅需求受到明显冲击，2023 年恢复速度显著提高；但相对于平稳趋势，游客和消费仍未完全恢复。数字经济通过连接文化供给和游客需求，有助于提升文化资源向文旅消费转化的效率。
