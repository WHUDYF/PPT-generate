## 一、选题背景与意义

本文选择“数字经济背景下全国博物馆数量增长与文旅产业复苏”作为研究主题。博物馆不仅是公共文化服务设施，也是城市文旅吸引力和文化消费的重要入口。随着文化数字化战略、智慧文旅平台、线上预约、数字展陈和文创电商的发展，博物馆的功能已经从线下参观扩展为“文化资源供给—数字化连接—文旅消费转化”的综合场景。

从国家宏观层面看，博物馆数量增长反映公共文化服务供给扩张，也体现文化强国建设和文旅融合发展方向。从民生角度看，更多博物馆意味着居民可获得的文化空间增加。从数字经济角度看，线上预约、智慧导览、数字展陈和平台传播能够提高文化资源与游客需求之间的连接效率。因此，本文选取博物馆数量、国内游客和国内旅游总花费三个指标，分析文化供给扩张与文旅产业复苏之间的关系。

本文重点回答三个问题：第一，博物馆数量是否保持长期增长？第二，2020 年后的文旅需求冲击能否通过数值方法识别？第三，2023 年文旅恢复是否已经回到正常增长趋势？

## 二、数据获取与预处理

数据主要来自国家统计局国家数据平台及《中国统计年鉴》公开指标。本文选取 2014-2023 年年度数据，博物馆数量单位为“个”，国内游客单位统一为“百万人次”，国内旅游总花费单位统一为“亿元”。

| 年份 | 博物馆数量 | 国内游客 | 旅游总花费 | 阶段 |
|---|---:|---:|---:|---|
| 2014 | 3658 | 3611 | 30311.9 | 增长期 |
| 2015 | 3852 | 4000 | 34195.1 | 增长期 |
| 2016 | 4109 | 4440 | 39390.0 | 增长期 |
| 2017 | 4721 | 5001 | 45660.9 | 增长期 |
| 2018 | 4918 | 5539 | 51278.3 | 增长期 |
| 2019 | 5132 | 6006 | 57251.0 | 高点 |
| 2020 | 5446 | 2879 | 22286.3 | 冲击 |
| 2021 | 5772 | 3246 | 29191.0 | 修复 |
| 2022 | 6565 | 2530 | 20444.0 | 波动 |
| 2023 | 6833 | 4891 | 49133.1 | 修复 |

预处理包括四步。第一，统一年份和单位，形成等距年度时间序列，采样间隔为 `h=1`。第二，保留 2020 年数据，因为它不是普通噪声，而是公共卫生事件造成的真实冲击。第三，以 2014 年为基期构造指数：

```text
指数 = 当年指标值 / 2014 年指标值 × 100
```

第四，构造转化效率指标：

```text
单馆游客量 = 国内游客 / 博物馆数量
人均旅游消费 = 国内旅游总花费 × 100 / 国内游客
单馆旅游消费 = 国内旅游总花费 / 博物馆数量
```

处理后发现，2019 年单馆游客量约为 1.170 百万人次/馆，2023 年为 0.716 百万人次/馆，说明供给持续扩张，但单馆游客承接量尚未恢复到高点；人均旅游消费从 2019 年约 953.2 元/人提高到 2023 年约 1004.6 元/人，说明消费强度恢复较快。

## 三、数值建模与算法实现

### 1. 三点差分

年度数据为等距离散点，本文使用三点中心差分计算年度边际变化：

```text
f'(x_i) ≈ [f(x_{i+1}) - f(x_{i-1})] / 2h
```

首尾年份采用单侧差分。核心代码如下：

```python
def three_point_diff(values, h=1.0):
    diff = []
    for i, value in enumerate(values):
        if i == 0:
            diff.append((values[1] - value) / h)
        elif i == len(values) - 1:
            diff.append((value - values[-2]) / h)
        else:
            diff.append((values[i + 1] - values[i - 1]) / (2 * h))
    return diff
```

计算结果显示，2020 年附近国内游客变化率约为 `-1380.0` 百万人次/年，2023 年游客变化率约为 `+2361.0` 百万人次/年，说明文旅需求在 2020 年受到明显冲击，而 2023 年恢复速度明显提高。

### 2. 分段最小二乘

考虑到 2020 年存在结构性冲击，本文不直接用一条直线拟合 2014-2023 年全周期，而是采用分段最小二乘。线性模型为：

```text
y = ax + b
```

目标是最小化残差平方和：

```text
S = Σ[y_i - (a x_i + b)]²
```

本文将数据分为 2014-2019 年平稳增长段和 2020-2023 年冲击后恢复段。模型比较结果如下：

| 指标 | 全周期线性 RMSE | 全周期二次 RMSE | 分段线性 RMSE |
|---|---:|---:|---:|
| 国内游客 | 1082.5 | 1017.3 | 432.8 |
| 旅游总花费 | 11968.2 | 11498.7 | 5118.5 |

可见分段线性模型误差明显更小。二次模型虽然样本内误差略低，但游客二次模型留一误差为 1747.7，高于线性模型的 1340.5，说明小样本下高阶模型外推不稳定。因此本文采用分段低次最小二乘。

### 3. 反事实缺口

为衡量恢复程度，本文用 2014-2019 年平稳趋势构造“无冲击情景”，计算 2020-2023 年实际值与趋势值的差距。

| 指标 | 2020 实际/趋势 | 2023 实际/趋势 |
|---|---:|---:|
| 国内游客 | 44.42% | 61.51% |
| 旅游总花费 | 35.81% | 62.42% |

结果说明，2023 年文旅需求明显恢复，但与正常增长趋势相比仍有缺口。

## 四、结果分析与价值阐释

从长期趋势看，博物馆数量从 2014 年的 3658 个增加到 2023 年的 6833 个，增长约 86.8%，说明公共文化供给持续扩张。与此同时，国内游客和旅游总花费在 2014-2019 年快速增长，但 2020 年显著下降，2023 年明显恢复。这体现了“供给侧持续扩张、需求侧受冲击后修复”的特征。

三点差分结果表明，2020 年游客变化率显著转负，2023 年变化率显著转正，说明文旅需求冲击和恢复都可以通过数值微分识别。分段最小二乘结果说明，面对存在外生冲击的社会经济数据，分段低次模型比全周期模型更合理。反事实缺口进一步说明，2023 年恢复不能简单理解为完全回归正常趋势，而应理解为“恢复速度提高，但仍未完全回到平稳增长轨道”。

从政策和数字经济角度看，博物馆数量增长提供了文化供给基础，但文旅复苏还需要数字化连接能力。线上预约、智慧导览、数字展陈、短视频传播和文创电商等手段，能够降低游客接触文化资源的成本，提高体验质量，并促进文化资源向消费场景转化。本文中人均旅游消费在 2023 年超过 2019 年，说明游客数量尚未完全恢复时，消费强度已经恢复较快，体现出文旅消费质量提升和转化效率改善。

本文也存在不足：年度采样间隔较大，无法反映月度波动；样本点只有 10 个，高阶拟合稳定性不足；博物馆数量与游客消费之间可能存在滞后效应。总体来看，本文通过三点差分、分段最小二乘和反事实缺口分析，解释了数字经济背景下文化供给扩张、文旅需求冲击和消费恢复之间的关系。
